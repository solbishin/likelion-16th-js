## 회고 (깨달은 점, 아쉬운 점 등)
- 어느정도 개념을 이해했다고 생각했는데 막상 문제가 나오니 아예 기억이 나지 않거나 이게 맞나? 라고 생각하는 부분이 많아 여러번 복습을 해야했다. 개념을 여러번 복습할 필요가 있다고 느꼈다. 
- 직접 코드를 계속 작성하다보니 계속 반복되는 부분이 많다고 느꼈다. 뭔가 반복하지 않는 방법을 고민하고 싶었는데 생각나는 부분이 없었다. 코드를 간결하면서 최대한 반복을 줄일 수 있는 방법을 생각해 봤는데 떠오르지 않았다. 어떻게 하면 간결한 코드를 쓸 수 있도록 생각 방식을 바꿀 수 있을지 고민이 된다.
