스타일 유틸리티 함수: getStyle, setStyle, css

자바스크립트에서 요소의 스타일을 쉽게 읽고, 설정할 수 있는 세 가지 함수 패턴을 알아봅니다. 반복되는 스타일링 코드를 효율적으로 관리하세요!

1. getStyle() 함수

getStyle(요소, CSS속성이름)은 요소의 계산된 스타일 값을 반환합니다. getComputedStyle()을 내부적으로 활용합니다. 

// 사용 예시
const element = document.querySelector('.target');
const colorValue = getStyle(element, 'color');
console.log(colorValue); // 'rgb(18, 137, 167)' 등

💡 Tip: 모든 적용 스타일 읽기

getStyle()은 인라인, 외부 CSS, 가상 요소 등 브라우저가 계산한 최종값을 반환합니다.

2. setStyle() 함수

setStyle(요소, CSS속성이름, CSS스타일값)은 요소의 인라인 스타일을 설정합니다. 요소 자체를 반환해, 체이닝이 가능합니다. 

// 사용 예시
const element = document.querySelector('.target');
setStyle(element, 'backgroundColor', '#f9f8f0');
setStyle(element, 'fontSize', '2rem');
// 반환값: element (수정된 요소)

⚠️ 주의: 인라인 스타일만 변경

setStyle()은 HTML 태그에 직접 스타일 속성을 추가합니다. 외부 CSS 파일의 값은 변경되지 않습니다.

3. css() 함수

css()는 읽기와 쓰기 모두 지원하는 통합 유틸리티 함수입니다. 

// 스타일 읽기
const color = css(element, 'color');
console.log(color); // 'rgb(18, 137, 167)'

// 스타일 설정
css(element, 'color', '#ff0000')
  .style.fontWeight = 'bold'; // 반환값: element (체이닝 가능)

💡 Tip: 읽기/설정 자동 분기

css()는 인자 개수에 따라 읽기/설정을 자동으로 처리합니다.

4. 함수 코드 예시

// getStyle 함수
function getStyle(element, property) {
  return getComputedStyle(element)[property];
}

// setStyle 함수
function setStyle(element, property, value) {
  element.style[property] = value;
  return element;
}

// css 함수
function css(element, property, value) {
  if (arguments.length === 2) {
    return getStyle(element, property);
  }
  setStyle(element, property, value);
  return element;
}

5. 요약 및 비교

스타일 유틸리티 함수 비교
함수명 기능 특징

setStyle 인라인 스타일 설정 수정 가능, 요소 반환
css 읽기/설정 통합 인자 개수에 따라 자동 분기, 체이닝 가능
