화살표 함수의 마법! 암묵적 반환 (Implicit Return)

함수 본문이 한 줄일 때 return 키워드 없이도 값을 반환하는 암묵적 반환의 규칙과 활용법을 배웁니다.

반환 방식 비교

반환 방식에 따른 차이점
항목 명시적 반환 (Explicit) 암묵적 반환 (Implicit)
중괄호 `{}` 사용함 (반드시 작성) 사용하지 않음
return 키워드 필요함 필요 없음 (자동 반환)
객체 반환 시 return { key: value } ({ key: value }) 괄호 필수

학습 포인트

  • 객체 반환의 핵심: 객체 리터럴을 반환할 때는 중괄호를 코드 블록으로 오해하지 않도록 소괄호 ()로 감싸야 합니다.
  • 코드 가독성: map이나 filter 같은 고차 함수와 결합할 때 코드가 훨씬 간결해집니다.
  • 조건부 반환: 비교 연산자(예: age >= 20) 자체가 true 또는 false를 암묵적으로 반환합니다.

실습 예제 미리보기

배열 데이터를 객체로 변환하거나, 조건에 맞는 성인 회원을 필터링하고, 과일 목록을 <li> 태그로 동적 렌더링하는 실습을 진행합니다.

    요약

    JavaScript 엔진에게 중괄호가 '함수의 몸통'인지 '데이터 객체'인지 정확히 알려주는 것이 중요합니다. 소괄호(())를 잊지 마세요!
