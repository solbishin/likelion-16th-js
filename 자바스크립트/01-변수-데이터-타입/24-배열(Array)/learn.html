Array 호텔 지배인 가이드

JavaScript 마을의 Array 호텔에 오신 것을 환영합니다. 지율 지배인이 꼭 알아야 할 호텔 운영 규칙(배열의 원리)을 정리했습니다.

1. 호텔의 시작: 0번 방의 법칙

Array 호텔의 가장 중요한 규칙은 첫 번째 방 번호가 0번이라는 점입니다.

지배인 메모: 손님이 3명일 때(length = 3), 마지막 손님은 3번 방이 아닌 2번 방(length - 1)에 계십니다.

2. 손님 목록(배열) 작성하기

대괄호([ ]) 방식
가장 대중적인 방법입니다. const cart = ['두부', '양파'];
생성자(Array) 방식
정석적인 객체 생성 방법입니다. new Array('두부', '양파');

3. 마스터 키(at)와 인덱스

특정 손님을 찾을 때는 주소(Index)를 사용하며, 최신 마스터 키인 at() 메서드를 사용하면 편리합니다.

배열 항목 접근 방법 요약
찾는 항목 전통적인 방식 최신 방식 (at)
마지막 항목 arr[arr.length - 1] arr.at(-1)
뒤에서 두 번째 arr[arr.length - 2] arr.at(-2)

⚠️ 지배인 금지 사항 (Anti-Pattern)

배열의 크기보다 훨씬 큰 인덱스에 값을 강제로 할당하지 마세요.

numberList[9] = '아홉' 처럼 중간 방을 비워두면, 스크린 리더는 비어있는 방들을 undefined로 읽게 되어 사용자에게 혼란을 줍니다.
배열 학습 내용이 화면에 표시되었습니다.
