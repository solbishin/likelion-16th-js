인테리어 전문가의 도구: style 속성 제어하기

호텔 방의 구조(HTML)를 잡았다면, 이제 벽지 색상이나 조명 밝기(CSS)를 자바스크립트로 동적으로 바꿔볼 차례입니다. style 속성은 실시간 인테리어를 위한 핵심 도구입니다.

1. CSS 스타일 설정의 기본

요소의 스타일을 직접 변경할 때는 style 속성에 새 값을 할당합니다. 브라우저가 지원하는 모든 CSS 속성에 접근할 수 있습니다.

const element = document.querySelector('.element');

element.style.color = 'rgb(237, 76, 103)';
element.style.backgroundColor = '#1289A7';
element.style.fontSize = '2.875rem';
element.style.fontWeight = '800';

⚠️ 주의: 카멜 케이스(Camel Case) & 단위

자바스크립트에서 style 속성을 사용할 때는 하이픈(-)을 제거하고 카멜 케이스로 작성해야 합니다.

  • user-select → userSelect
  • margin-block-start → marginBlockStart

또한, 크기를 지정할 때는 반드시 문자열로 단위(px, rem, %)를 포함해야 정상 작동합니다.

2. 유연한 스타일 제어: setProperty()

setProperty() 메서드는 더 세밀한 제어가 필요할 때 사용합니다. 특히 CSS 변수(사용자 정의 속성)를 다룰 때 필수적입니다.

// 일반 속성 설정
요소.style.setProperty('background-color', '#A3CB38');

// 사용자 정의 속성(CSS 변수) 설정
요소.style.setProperty('--theme-color', '#A3CB38');

지배인 메모: setProperty를 쓸 때는 자바스크립트 방식(카멜 케이스)이 아니라 CSS 본래의 방식(하이픈 케이스)을 그대로 사용해야 한다는 점을 잊지 마세요!

3. 스타일 제어 방법 요약

스타일 설정 방식 비교
구분 문법 형태 특징 및 설명
style 속성 요소.style.속성명 = '값' 가장 흔한 방식, 카멜 케이스 사용
브라켓 표기법 요소.style['속성명'] 속성명을 변수로 동적 할당할 때 유용
setProperty() style.setProperty('명', '값') 하이픈 케이스 사용, !important 설정 가능
사용자 정의 속성 setProperty('--명', '값') CSS 변수 제어 시 필수 방식
