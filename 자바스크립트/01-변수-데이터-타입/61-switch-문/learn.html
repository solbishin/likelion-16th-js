분기 제어: switch 문

복잡한 if...else if 구조를 특정 값에 따라 명확하게 나누는 switch 활용법을 배웁니다.

switch 문 구조 요약

switch 문 구성 요소
구성 역할 비고
case '값': 입력값과 일치하는지 확인 엄격한 비교(===) 수행
break 블록 실행 중단 및 탈출 생략 시 다음 case까지 실행(Fall-through)
default: 일치하는 값이 없을 때 실행 else와 유사한 역할
