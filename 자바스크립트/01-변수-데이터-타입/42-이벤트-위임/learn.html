개별 요소가 아닌 상위 요소에 단 하나의 리스너를 등록하여 다수의 하위 요소를 효율적으로 제어하는 기법을 배웁니다.

이벤트 위임의 작동 원리

  • 버블링 활용: 하위 요소에서 발생한 이벤트는 부모 요소로 전파됩니다.
  • 대상 확인: e.target을 통해 실제 클릭된 지점을 파악합니다.
  • 조건 검사: matches()나 closest()로 원하는 대상인지 확인합니다.

중첩된 요소 처리 방법

링크(a) 내부에 아이콘이나 텍스트가 있을 때, 항상 링크 요소가 선택되도록 보장하는 방법입니다.

중첩 요소 클릭 문제 해결 방식 비교
방법 기술 특징
JavaScript closest('a') 가장 가까운 조상 링크를 찾아 안전하게 참조
CSS pointer-events: none 자식 요소가 마우스 이벤트를 무시하고 통과시킴

요약

이벤트 위임은 성능 최적화와 동적 UI 구현을 위한 필수 기법입니다. e.target과 closest()의 조합을 잘 활용하는 것이 핵심입니다.
