DOM 동적 제어: 제거와 이동

요소를 스스로 삭제하는 remove와 위치를 재배치하는 이동 기법을 배웁니다.

제거 메서드 비교

삭제 및 이동 메서드 요약
메서드 설명 특징
removeChild() 부모로부터 자식 제거 제거된 요소를 반환함
remove() 요소 스스로 삭제 최신 방식, 반환값 없음
insertBefore() 특정 위치로 이동 기존 위치에서 자동 제거됨
