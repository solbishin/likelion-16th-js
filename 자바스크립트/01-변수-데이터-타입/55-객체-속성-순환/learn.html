객체를 배열처럼! Object Iteration

순서가 없는 객체(Object)를 배열(Array)로 변환하여 데이터를 자유자재로 순회하는 기법을 배웁니다.

Object 정적 메서드 삼총사

객체 변환 메서드 비교
메서드 결과물 (Array) 활용 상황
Object.keys() ['key1', 'key2'] 속성 이름(키)만 나열하거나 존재 여부 확인할 때
Object.values() [value1, value2] 데이터의 총합, 평균 등 수치 계산이 필요할 때
Object.entries() [['k1', v1], ['k2', v2]] 키와 값을 동시에 활용하여 순회할 때 (가장 권장)

학습 포인트

  • 순회 가능성: 객체는 직접 반복할 수 없지만, 배열로 바꾸면 for...of나 forEach 사용이 가능합니다.
  • 구조 분해 할당: entries() 사용 시 [key, value] 형태로 바로 받는 것이 가독성에 매우 좋습니다.
  • 동적 제어: 객체의 속성이 몇 개인지 모를 때 유용하게 전체 데이터를 훑을 수 있습니다.

요약

객체 데이터를 화면에 리스트 형태로 뿌려야 할 때 Object.entries()는 필수입니다. 배열의 강력한 메서드들과 객체의 구조적인 장점을 결합해 보세요!
