데이터를 간결하게! 구조 분해 할당

객체나 배열의 복잡한 구조를 분해하여 필요한 값만 변수로 추출하는 효율적인 문법을 배웁니다.

구조 분해의 두 가지 형태

객체와 배열 구조 분해 비교
구분 객체 (Object) 배열 (Array)
사용 기호 { } (중괄호) [ ] (대괄호)
할당 기준 속성 이름 (Key) 데이터 순서 (Index)
주요 특징 이름 변경(:) 가능 건너뛰기(, ,) 가능

고급 활용 기법

  • 기본값 설정: const { a = 10 } = obj 형식으로 데이터 누락에 대비합니다.
  • 매개변수 분해: 함수 인자로 넘어온 거대한 객체에서 필요한 것만 즉시 추출합니다.
// 함수 매개변수 즉시 분해 예시
function display({ name, age }) {
  console.log(`${name}(${age})님 환영합니다!`)
}
💡 클린 코드 팁

구조 분해 할당을 사용하면 props.user.profile.name 처럼 길어지는 참조를 단 한 번의 선언으로 줄일 수 있습니다. 이는 코드의 가독성을 높일 뿐만 아니라 실수를 줄여주는 아주 좋은 습관입니다!

요약

구조 분해 할당은 불필요한 반복을 제거하고 데이터의 구조를 직관적으로 파악하게 해줍니다. 객체는 이름으로, 배열은 순서로 분해한다는 점만 꼭 기억하세요!
