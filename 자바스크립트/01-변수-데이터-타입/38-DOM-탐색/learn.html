DOM 탐색 (Traversal)

기준 요소에서 부모, 자식, 형제 요소로 직접 이동하여 원하는 대상을 효율적으로 선택하는 방법을 배웁니다.

탐색 방향별 주요 메서드

방향에 따른 DOM 탐색 속성 및 메서드 요약
방향 주요 속성 및 메서드 특징
아래로 (자식) children, firstElementChild 직계 자식 요소 탐색
위로 (부모) parentElement, closest() 부모 또는 가장 가까운 조상 탐색
옆으로 (형제) nextElementSibling, previousElementSibling 이전/다음 형제 요소 탐색

방어적 프로그래밍

탐색 결과가 존재하지 않을 경우 null을 반환하므로, 반드시 존재 여부를 확인하는 습관이 중요합니다.

if (element.closest('.container')) { /* 로직 실행 */ }
💡 핵심 팁
  • children은 HTMLCollection을 반환하므로 순회 시 배열 변환이 필요할 수 있습니다.
  • item() 메서드는 범위를 벗어날 때 null을 반환하여 안전한 에러 처리를 돕습니다.
