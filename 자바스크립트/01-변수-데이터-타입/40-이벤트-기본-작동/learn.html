브라우저 기본 작동 방지

링크 이동이나 폼 제출과 같이 브라우저가 기본적으로 수행하는 고유 동작을 제한하고 자바스크립트로 제어하는 방법을 배웁니다.

주요 기본 동작 예시

  • 링크(<a>): href 주소로 페이지 이동
  • 폼(<form>): 데이터 전송 후 페이지 새로고침(리디렉션)
  • 체크박스: 클릭 시 선택/해제 상태 전환

preventDefault() 메서드

이벤트 객체의 e.preventDefault()를 호출하면 브라우저의 기본 동작이 실행되지 않습니다.

기본 작동 방지 관련 속성 요약
항목 설명
실행 메서드 e.preventDefault()
방지 여부 확인 e.defaultPrevented (방지 시 true)
