논리 연산자로 조건 처리

복잡한 여러 조건을 && (AND)와 || (OR) 연산자로 연결하여 코드를 간결하고 명확하게 만드는 방법을 배웁니다.

논리 연산자 핵심 비교

AND 연산자와 OR 연산자의 작동 방식 비교
구분 && (AND 연산자) || (OR 연산자)
의미 "그리고" (모두 참이어야 참) "또는" (하나만 참이어야 참)
평가 중단 지점 거짓을 만났을 때 참을 만났을 때
반환 값 첫 번째 거짓 값 또는 마지막 값 첫 번째 참 값 또는 마지막 값

단락 평가 (Short-circuit Evaluation)

JavaScript는 효율성을 위해 결과가 결정되는 즉시 나머지 평가를 생략합니다.

// userName이 없으면 '익명'을 할당 (OR 활용)
const name = userName || '익명';

// user가 존재할 때만 프로필 출력 (AND 활용)
user && console.log(user.profile);

요약

논리 연산자는 단순한 참/거짓 판별을 넘어 값의 선택과 실행 제어에도 널리 쓰입니다. 단락 평가의 원리를 이해하면 훨씬 세련된 코드를 작성할 수 있습니다.
